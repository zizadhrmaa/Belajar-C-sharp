== Bikin file solution ==
dotnet new sln -n <nama_file>

== Set-up dan bikin Folder dalam folder root ==
dotnet new console -o <nama-folder> 

== Tambahkan ke solution ==
dotnet sln add <nama_folder>/<nama_folder>.csproj


== Cara run ==
dotnet run --project <nama-folder>
